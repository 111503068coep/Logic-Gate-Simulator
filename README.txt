Title of project: Logic Gate Simulator
Name: Vaishnavi Bobde
MIS: 111503068
Description of project:
library used: Gtk

work done is:

a gtk window is created
a toolbox that contains buttons of gates, wires and inputs and output with RUN and STOP buttons
an icon for logic gate simulator
drag and drop function for all gates, inputs and output
RUN button to get the output, STOP button to stop the output
a menubar is added having options:
1. new
2. save
3. open
4. quit
open menu opens an image in the window
quit menu quits the program
the gates are stored in an array of integers
and then output is traced that is depicted when RUN is clicked
